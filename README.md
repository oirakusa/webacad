## Іменування Git branch

Структура іменування git branch:  
`firstName_lastName/homeworkNumber_homeworkTheme`  
For example:  
`oleksandr_sapozhnikov/04_terragrunt`

## Іменування Pull requests

Структура заголовка "Pull request":  
`[firstName_lastName]: homeworkNumber-homeworkTheme`  
Приклад:  
`[oleksandr_sapozhnikov]: 04-terragrunt`

У полі **Reviewers**, вказуємо тренера який викладав тему

## Як відправити посилання на дз?

1. Створіть текстовий файл “Course_Surname_HW\_\_.txt” у себе на робочій машині
2. Додайте у файл посилання на свій репозиторій або посилання на Pull request з вирішенням даного завдання, збережіть файл
3. Завантажте збережений файл у відповідь на це завдання

## Як відправити ДЗ файлом?
1. Додайте до папки матеріали з вирішенням цього завдання, збережіть файл
2. Заархівуйте файл з розв’язанням цього завдання під назвою “Course_Surname_HW\_\_.zip” у себе на робочій машині
3. Завантажте збережений файл у відповідь на це завдання
